# Shadow Hunter

**Autonomous Agentic Discovery of Hidden AI & Cloud Dependencies**

Shadow Hunter is an enterprise-grade security platform designed to detect unauthorized "Shadow AI" agents and services within cloud environments.

## Features

- **Passive Detection:** Analyze network traffic to identify AI communication patterns.
- **Active Interrogation:** Safely probe unknown services to determine their nature.
- **Graph-Based Viz:** Visualize real-time dependencies and data flows.
- **AI-Powered Analysis:** Use behavioral models to detect anomalies and policy violations.

2.  **Install Dependencies:** `pip install -e .`
3.  **Run Services:** See specific service READMEs in `services/`.
